육아 특화 감정 분석 가이드라인

== 육아 단계별 주요 감정 ==
👶 신생아기 (0-3개월): 불안, 피로, 책임감
🍼 영아기 (4-12개월): 성취감, 걱정, 애착
👶 유아기 (1-3세): 스트레스, 자부심, 좌절
🎒 학령전기 (4-6세): 기대, 경쟁심, 우려

== 상황별 감정 패턴 ==
수면: 피로 → 짜증 → 절망 → 체념
식사: 걱정 → 스트레스 → 포기 → 수용  
발달: 기대 → 불안 → 비교 → 수용
사회성: 자부심 → 걱정 → 비교 → 지지추구

== 지원 필요 시그널 ==
- 고립감 표현 → 커뮤니티 연결 필요
- 반복적 걱정 → 전문가 상담 연결
- 피로 호소 → 실용적 도움 매칭
- 비교 불안 → 같은 상황 부모 매칭
